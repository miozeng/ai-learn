### opencv
opencv的内容来自于网易云的课程--opencv计算机视觉实战

主要包括

|知识点|说明|  
|---|---| 
|[1.图像的基本操作](1.图像基本操作.ipynb)| 图片与视频的读取，颜色通道，边界填充，推按截取与融合|
|[2.图像的形态学](2.图像形态学.ipynb)|图像阈值（ 当像素值高于阈值时，我们给这个像素赋予一个新值，否则给他赋予另一个值）；平滑（模糊，处理噪音）态学操作用于图像预处理操作(去噪 形状简化) 图像增强(骨架提取 细化 凸包 物体标记) 物体背景分割及物体形态量化等场景|
|[3.图像梯度与边缘检测与模板匹配](3.图像梯度与边缘检测与模板匹配.ipynb)|图像的梯度（提取图像的边缘）Canny边缘检测，图像金字塔，图像轮廓，模板匹配|
|[4.直方图与傅里叶变化](4.直方图与傅里叶.ipynb)|直方图（利用直方图可以对图片的均衡化（对比度）进行调整，包括局部调整），mask操作，直方图均衡化，傅里叶变换 （空间域转换为频域高频：变化剧烈的灰度分量，例如边界，低频：变化缓慢的灰度分量，例如一片大海）|
|[实战：文档扫描与OCR识别](filescan.py)|图片处理得到更可读的小票文件，然后通过ocr工具读取小票|
|[5.图像的特征](5.图像特征.ipynb)|特征的读取（角点检测斑点检测），特征匹配|
|[实战：全景图片拼接](picSplicing.py)|图片的拼接|
|[6.背景建模与光流估计](6.背景建模与光流估计.ipynb)|帧差、中值滤波、高斯平均、混合高斯、Vibe和以纹理为特征的背景建模，光流是空间运动物体在观测成像平面上的像素运动的“瞬时速度”，根据各个像素点的速度矢量特征，可以对图像进行动态分析，例如目标跟踪|
|[实战：目标跟踪](multi-object-tracking.py)|利用OpenCV 中的 MultiTracker 实现目标跟踪|
|[实战：人脸识别](face_recognition)|各种人脸识别算法的实践，包括眨眼验证|
